Mendeley

Mendeley is a reference manager and academic social network.

Mendeley uses OAuth2 authentication, we automatically obtain the access token using the client_id and secret.

Required configuration fields

  • client_id and client_secret (secret): register your OAuth2 application at http://dev.mendeley.com
ALM Name mendeley
ALM Configuration default
ALM Core Attributes
(for groups only)		 date (as date_added)
contributor (as profile_id)
ALM Other Attributes group_id
Protocol REST
Format JSON
Rate-limiting 150/hour
Authentication OAuth 2.0
Restriction by IP Address no
API URL https://api.mendeley.com/catalog?doi=DOI&view=stats
