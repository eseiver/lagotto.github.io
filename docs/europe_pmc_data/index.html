Europe PubMed Central (Europe PMC) is an archive of life sciences journal literature. Europe PubMed Central tracks the database entries that cite a given publication.

ALM Name europe_pmc_data
ALM Configuration default
ALM Core Attributes count
ALM Other Attributes database name
Protocol REST
Format JSON
Rate-limiting unknown
Authentication no
Restriction by IP Address no
API URL http://www.ebi.ac.uk/europepmc/webservices/rest/MED/PMID/databaseLinks//1/json
