Europe PubMed Central

Europe PubMed Central (Europe PMC) is an archive of life sciences journal literature. Europe PubMed Central tracks the citations for a given publication.

ALM Name europe_pmc
ALM Configuration default
ALM Core Attributes count
ALM Other Attributes none
Protocol REST
Format JSON
Rate-limiting unknown
Authentication no
Restriction by IP Address no
API URL http://www.ebi.ac.uk/europepmc/webservices/rest/MED/PMID/citations/1/json
