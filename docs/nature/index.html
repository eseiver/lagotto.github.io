Nature

Nature Blogs is a science blog aggregator. Since May 1st 2013 API keys are no longer necessary.

ALM Name nature
ALM Configuration staleness: [ 1.day, 1.month, 1.month, 1.month]
requests_per_hour: 200
ALM Core Attributes id
url
date (as published_at)
ALM Other Attributes title
blog title
blog url
Protocol REST
Format JSON
Rate-limiting 2/sec
5,000/day
Authentication no
Restriction by IP Address no
API URL http://blogs.nature.com/posts.json?doi=DOI&api_key=API_KEY
License CC-BY-NC
